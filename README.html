Readme For the Knowledge and Data Project

There are two ontologies avaliable

  • One without inferences (final_ontology_without_inferences.ttl)
  • Second with inferences (final_inferred_ontology.ttl)

You should add the ontology with inferences to GRAPHDB!

To display the map you should trust the Notebook!

Makes sure that you have the following python packages installed

  • Pandas
  • Folium
  • SPARQLWrapper

We recommend running the notebook inside a browser for the best experience.
